# CSVFromGmailToSheets
Automatiza la importación de datos en formato CSV desde correos electrónicos específicos en Gmail a una hoja de cálculo de Google Sheets. Utiliza un término de búsqueda en Gmail para encontrar el correo electrónico deseado, extrae el archivo CSV adjunto si existe, y luego importa los datos de ese archivo CSV a la hoja de cálculo en Google Sheets.
