# CSV from Gmail to Sheets
Las empresas están tendiendo a eliminar los paquetes Office (local) y mover todo al cloud, en este caso me encontraba con el problema de subir y actualizar informes en Google Sheets debido a la cantidad de filas que tenían los informes, por eso nace este script para automatizar esas actualizaciones y subidas de archivos desde Gmail que es por donde nos entraba la nueva información mensualmente.

Este script tiene el objetivo de automatizar la importación de datos en formato CSV desde correos electrónicos específicos en Gmail a una hoja de cálculo de Google Sheets. Utiliza un término de búsqueda en Gmail para encontrar el correo electrónico deseado, extrae el archivo CSV adjunto si existe, y luego importa los datos de ese archivo CSV a la hoja de cálculo en Google Sheets.
